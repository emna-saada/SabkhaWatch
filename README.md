# Sebkha Watch

Sebkha Watch is a website prototype for an ecological and geomatics platform that helps monitor Tunisian sabkhats.

The project combines environmental protection with interactive mapping. Its goal is to make problems like pollution, drying, and illegal dumping easier to see, report, and manage.

## Main Idea

Many sabkhats in Tunisia, such as Sabkhat Sidi El Hani, Sabkhat El Kelbia, and Sabkhat Ariana, are fragile ecosystems.

Today, information about these places is often scattered between researchers, citizens, municipalities, and reports. Sebkha Watch brings this information together in one map-based platform.

## Main Features

- Interactive map with colored environmental status pins
- Citizen report form for pollution or field observations
- Project section for ecological restoration ideas
- Startup storytelling section for innovative solutions
- Simple dashboard showing alerts, reports, and monitored sabkhats

## Users

The platform is designed for:

- Researchers who need environmental data
- Citizens and volunteers who want to report problems
- Municipalities that need to respond to alerts
- Investors or donors who want to support ecological projects

## UI/UX Design

The design is simple and clean.

Main design choices:

- The map is the main element of the website
- Green, orange, red, and blue are used to show status clearly
- Cards are used to organize information
- Buttons and navigation are easy to understand
- The layout is responsive for different screen sizes

## Technologies Used

- HTML
- CSS
- Responsive design
- No JavaScript
- No backend

## How to Open the Project

1. Download the project files.
2. Open the HTML file in a web browser.
3. Use the navigation menu to visit:
   - Map
   - Reports
   - Projects
   - Pitch Zone
